# Student Information Management System - Web Version

This is the web-based version of the Student Information Management System that runs on GitHub Pages.

## Features

- **Login System**: Role-based authentication (Admin, Teacher, Student)
- **Student Management**: View student details (double-click to view full details)
- **Data Storage**: Uses browser localStorage (similar to data.json in desktop version)

## Default Login Credentials

- **Admin**: `admin` / `admin123`
- **Teacher**: `teacher` / `teacher123`
- **Student**: `student` / `student123`
